# Auto Electrical Wiring Planner

## Objective

Build a web app that:

1. Lets users draw a floor plan
2. Extracts coordinates from the drawing
3. Generates an electrical wiring layout automatically
4. Renders the result in 2D (SVG) and optionally 3D

---

## Core Idea

Blueprint → Structured Data → Layout Engine → Visual Output

---

## Key Features

- Interactive drawing tool
- Automatic outlet and switch placement
- Intelligent wire routing
- AI-assisted optimization
- Visual output (SVG / optional 3D)

---

## Scope

Focus on:

- Simple layouts
- Heuristic-based placement
- Visual clarity

Avoid:

- Complex blueprint parsing
- Real electrical compliance
